# LMSCloud Koha MahnStudio

MahnStudio ist ein Editor für Koha-Mahnungsvorlagen (HTML/CSS) mit Live-Vorschau. 

## Installation

**Voraussetzung:** [Node.js](https://nodejs.org/) (LTS) und npm.

```bash
# Repository klonen
git clone <repo-url>
cd LMSCloudKohaMahnStudio

# Abhängigkeiten installieren
npm install
```

**Entwicklung starten:** `npm run dev` — App läuft lokal (z. B. http://localhost:5173).

**Produktionsbuild:** `npm run build` — Ausgabe in `dist/`. Mit `npm run preview` den Build lokal ansehen.

Unter **Windows** die gleichen Befehle in PowerShell oder „Node.js command prompt“ ausführen. Unter **macOS/Linux** in einem Terminal.
